Input: DOM pointer and gesture event handling sample

This sample shows how to use MSPointerEvent and MSGestureEvent objects with various Document Object Model (DOM) events in a Metro style app using JavaScript.

Specifically, this sample covers:

  • Using the HTML 5 Canvas element as a 2-D drawing surface.
  • Implementing drawing functionality with JavaScript and Cascading Style Sheets (CSS).
  • Handling interactions from touch, mouse, and pen/stylus input devices through the following DOM events:
    • onmspointerdown
    • onmspointermove
    • onmspointerup
    • onmspointerover
    • onmspointerout
    • onmsgesturedoubletap

Build the sample

To build this sample:

  1. Start Visual Studio and select File > Open > Project/Solution.
  2. Go to the directory in which you unzipped the sample. Go to the directory named for the sample, and double-click the Microsoft Visual Studio Solution (.sln) file.
  3. Press F6 or use Build > Build Solution to build the sample.

Run the sample

To debug the app and then run it, press F5 or use Debug > Start Debugging. To run the app without debugging, press Ctrl+F5 or use Debug > Start Without Debugging.
